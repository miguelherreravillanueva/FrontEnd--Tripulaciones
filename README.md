## Objectives
### The main objective of this project has been to meet the needs and pay attention to the requirements of the Campanar Neighborhood Association of Valencia (Spain).
### This project has been carried out from scratch together with the UX/UI, Cybersecurity and Data Analyst departments for a period of 2 weeks.
### In this project users can: 
<objectives>
  <ol>
    <li>Register</li>
    <li>Log in</a></li>
    <li>View profile</a></li>
    <li>Create requests</a></li>
    <li>Add comments</a></li>
</ol>
</objectives>

### And so de admin can: 
<objectives>
  <ol>
    <li>Create activities</a></li>
    <li>Create news</a></li>
</ol>
</objectives>

## About The Project

In this project, a social network is developed where users will register, log in, view their profile and create requests and comments using the and own API created for this same project <a href="https://github.com/miguelherreravillanueva/Desafio-Tripulaciones-Back"></a>,
